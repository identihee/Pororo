### 서비스 개요
#### 왜 이 서비스를 만들었나 - 타 서비스와 차별점
- “오늘은 무조건 끝낼 거야!” 라고 다짐해도 스마트폰, 숏폼 콘텐츠 때문에 집중이 깨져 미뤄둔 일만 산더미 생각만 해도 스트레스 받습니다.
- 기존의 뽀모도로/타이머 앱은 시간 측정에 그칠 뿐 **지속적인 동기 부여**를 제공하지 못합니다.
- 특히 번아웃이나 동기 저하 상태에서는 작은 성취가 즉시 보상으로 이어지지 않아 꾸준함을 유지하기 어렵습니다.
- 삶의 습관 개선으로 인한 스트레스 감소, 생산성 향상, 번아웃 예방에 기여 — 교육·기업 복지 시장에서도 확장 가능합니다.

#### 우리의 인사이트
- 게임은 즉각적인 피드백과 보상으로 사용자의 몰입을 이끌어냅니다.
- 공부·운동·업무의 집중 시간도 게임 내 스탯처럼 **즉시 가시화**하면 동기 유지에 효과적입니다.
- **Review of Research on Student-Facing Learning Analytics Dashboards**
[![Paper](https://img.shields.io/badge/Paper-PDF-red)](https://ieeexplore.ieee.org/document/8010828)
*Bodily, R., & Verbert, K. (2017)*와 같이 시각화된 수치가 학습자의 동기와 성취도에 긍정적인 영향을 미친다는 주장은 교육학 분야에서 널리 입증된 사실입니다.
- AI로 개인화된 타이머를 제공하면 ‘효율적인 집중 습관’이 더 쉽게 만들어집니다.
- 현재 핵심 기능(타이머, 세션 기록, 기본 성장 시스템)을 구현했고, React + Flask 기반으로 빠른 개선과 확장이 가능합니다.

### 구현 내용 및 결과물
#### 핵심 솔루션
- AI 기반 개인화: 사용자의 세션 데이터인 달성률을 분석해 활동별 최적 집중/휴식 주기를 추천
- 실시간 뽀모도로 타이머: 시작/종료 알람, 세션 기록, 통계 출력
- 메인 성장 미션 : 집중/휴식 성과에 기반한 미션 달성과 보상.

#### 주요 기능 상세

1) 테마별 집중/휴식 추천 (AI 코칭)
  - 활동 테마(공부/일/운동 등)에 맞춘 권장 집중 시간과 휴식 주기를 자동 제안
  - 규칙 기반 + 사용자 누적 데이터(일/주/월 평균 달성률)를 결합한 분석 엔진

2) 실시간 타이머 & 성과 추적
  - 뽀모도로 방식의 타이머(집중/휴식 알람, 중단 감지)
  - 세션 기록 저장
  - 세션 종료 시 집중 달성도에 따른 스탯 변화(근력/지력/피로도 등)

3) 캐릭터 & 공간 성장
  - 성취가 시각적으로 즉시 보상되어 작은 성공들이 누적되는 경험 제공

#### 기술 스택
- 프론트엔드: JavaScript, React
- 백엔드: Flask (Python)
- 데이터: 로컬 DB 연동
- AI: 사용자 로그 기반 규칙기반

로컬 실행 가이드 (개발용)
- 프론트엔드
  ```
  cd ai-pomodoro-mvp
  npm install
  npm run start
  ```
- 백엔드
  ```
  cd backend
  python -m venv .venv
  source .venv/bin/activate   # Windows: .venv\Scripts\activate
  pip install -r requirements.txt
  flask run
  ```
- 환경 변수, API 키, DB 설정은 각 폴더의 README 또는 .env.example 참고
